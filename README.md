# ClinIQ 🧬 — AI Medical Case Generator

> An AI-powered synthetic medical case generator built for students, educators, and researchers.

##Live Demo
https://medical-case-generator.netlify.app/

## What is ClinIQ?

ClinIQ generates realistic synthetic medical case studies on demand using LLMs. 
It evolved from my Medical Case Quiz Simulator project. I identified that static, 
fixed cases were a key limitation, and built ClinIQ to solve that using Generative AI.

Instead of memorizing fixed cases, students can now generate unlimited, 
unique clinical scenarios and practice diagnostic reasoning interactively.

## Features
-  6 Medical Specialties — Cardiology, Neurology, Pulmonology, Gastro, Endocrine, Infectious Disease
-  3 Difficulty Levels — Beginner, Intermediate, Advanced
-  Diagnostic Reveal Mechanic — diagnosis and treatment are hidden until you're ready
-  Synthetic Data Generation — unlimited unique cases every time
-  Powered by Groq LLM API (llama-3.3-70b-versatile)


## Tech Stack
- React.js + Vite
- Groq API (LLM Integration)
- CSS-in-JS animations

## Setup

1. Clone the repo
   git clone https://github.com/Namitha906/cliniq.git
   cd cliniq

2. Install dependencies
   npm install

3. Create a .env file in the root folder
   VITE_GROQ_API_KEY=your_groq_api_key_here

4. Run the app
   npm run dev

5. Open http://localhost:5173
